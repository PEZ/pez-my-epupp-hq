## Design Context

### Users
PEZ — power-user Clojure developer, creator of Calva. Uses Epupp to live-tamper web pages via REPL. The UI is for one person who values density, speed, and functional elegance.

### Brand Personality
Sharp, playful, confident. The gold lightning bolt on blue is the identity — unmistakably Epupp, proudly injected into host pages. Not trying to hide or blend in. A hacker's tool that looks like it belongs in a heist movie's control room.

### Aesthetic Direction
- **Theme**: Dark — always. Epupp lives in dark panels against any host page.
- **Palette**: Gold accent (#FFDC73), Epupp blue (#4A71C4), dark surfaces (#1e1e1e, #2d2d2d). Neutrals tinted warm toward the gold.
- **Density**: Compact. Tight rows, lots of information visible. Power-user density.
- **Typography**: Small, precise, monospace for data, sans for labels. No decorative fonts.
- **Anti-references**: Generic browser extension popups, Tampermonkey settings panels, bloated dev tools with too many tabs. Should feel like a scalpel, not a Swiss army knife.

### Design Principles
1. **Proud injection** — Epupp UI should look intentionally placed, not hacked in. The gold border is a signature, not a placeholder.
2. **Information-dense** — Show maximum useful data in minimum space. No padding for padding's sake.
3. **Functional playfulness** — The lightning bolt icon, the gold flashes, the confident placement say "I own this page now."
4. **Contextual intelligence** — UI adapts to where it's placed (below video, beside controls) but never loses its Epupp identity.
5. **Keyboard-first** — Enter to search, Escape to close, typing just works. Mouse is secondary.
