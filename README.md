# PEZ's Epupp HQ

My personal workspace for [Epupp](https://github.com/PEZ/epupp) — live web tampering, userscript development, and accumulated site-specific helpers. Based on the [my-epupp-hq](https://github.com/PEZ/my-epupp-hq) template.

This project grows with use. Userscripts, live-tamper helpers, and configuration accumulate here as I work with Epupp day to day. General improvements discovered here get contributed back to the template.

## Structure

- **`userscripts/`** — Scripts with manifests, synced to Epupp via `bb` tasks
- **`live-tampers/`** — REPL-evaluated code, organized per site (`github/`, `youtube/`, etc.)
- **`snippets/`** — Reusable hiccup and code fragments
- **`docs/`** — Epupp documentation synced from upstream (`bb docs-sync`)
